# gutentag-world-nim

> Gutentag, World! in Nim

## Run

```
nim compile --run src/gutentag_world.nim
```


## Install

```
nimble install gutentag_world
```


### Usage

## Usage

```sh
nimble run
```

Or import in your project:

```nim
import gutentag_world

echo gutentagWorld()
```
